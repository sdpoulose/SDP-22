Opioid Epidemic Mapping

A predictive look into the United States opioid crisis

As part of the University of Connecticut School of Engineering Senior Design Program, this project was sponsored by Cigna. The results of the project are exhibited in a series of predictive models estimating opioid death rate based on a number of social determinants.
For information about each map, see below.

State Map

Shown above are real opioid death rates from 2016 given by the CDC, as well as predicted death rates per state for the year 2020. The map was generated using a k-nearest neighbor machine learning algorithm using five social determinants.



2013 County Map

Shown above are real and generated opioid death rates for 2013. The real values were given by the CDC, and the predicted values were generated from a random forest machine learning algorithm using four social determinants. The model was trained on data from years 2011 and 2012.
The years spanning 2009-2012 were used as training data and 2013 was the testing data. Compared to other subsets of data, this subset was found to have the best results. It is likely because the input variables were potentially more correlated to the death rate in earlier years than later years. The possible implication is that the opioid is more far-reaching in recent years, contributing to the increasingly high death rate.



2018 County Map

Shown above are real and generated opioid death rates for 2018. The real values were given by the CDC, and the predicted values were generated from a random forest machine learning algorithm using four social determinants. The model was trained on data from years 2016 and 2017.
The years spanning 2016-2017 were used as training data and 2018 was the testing data.
